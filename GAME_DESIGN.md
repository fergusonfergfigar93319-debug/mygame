# 咕咕嘎嘎 游戏设计稿

## 项目概述

`咕咕嘎嘎` 是一款原创单机横版动作冒险游戏，核心体验围绕：

- 企鹅童话风格
- 伙伴重聚
- 家园重建
- 轻动作 + 轻探索 + 轻剧情

游戏不是对经典平台玩法的照搬，而是围绕“被毁掉的家园、失散的伙伴、逐步成长的旅程”展开。

## 核心卖点

1. 有明确主线和角色关系，不只是闯关
2. 道具设计围绕企鹅世界观展开
3. 伙伴会逐步加入并提供不同能力
4. 家园重建会反向推动剧情和成长

## 当前原型已落地内容

- 第一关 `雪松村废墟`
- 主角 `咕咕嘎嘎`
- 反派 `高松鹅`
- 基础移动、跳跃、平台、坑洞
- `鱼干冲刺`
- `泡泡围巾`
- `团团支援`
- 手感：离地后短「土狼时间」内仍可起跳；长按跳跃键可跳得略高（早松手会缩短上升）
- 表现：远景飘雪与日光、关卡金币轻微浮动、关底旗帜与伙伴区域层次更清晰；HUD 含路程进度条与道具/伙伴状态标签
- 计分：拾取鱼干、踩踏敌人会累计额外奖励分（与金币、路程分合并计入总分与最高分）

## 推荐功能路线

### 第一阶段：玩法成型

- 完成第一关
- 接入第二关入口
- 稳定伙伴支援系统

### 第二阶段：内容扩展

- 第二关 `冰湖回音谷`
- 新伙伴能力
- 家园修复据点

### 第三阶段：系统完善

- 正式开始界面
- 章节选择
- 存档与进度系统

## 道具系统方向

### 鱼干冲刺

- 提高短时间移动速度
- 适合高速穿越和撞开脆弱地形

### 泡泡围巾

- 提供缓降效果
- 用于扩大地图垂直空间玩法

### 雪团手套

- 用于冻结机关或击退敌人

### 热汤壶

- 用于解谜、融冰和开启隐藏路径

## 伙伴能力方向

### 团团

- 当前定位：一次性战斗支援
- 后续可扩展：推动重物、压住开关、帮忙破坏障碍

### 泡泡

- 负责水域和潜水关卡

### 啾啾

- 负责滑翔和高空路线

### 阿绒

- 负责风雪天气和特殊视野区域

## 关卡设计原则

每一关尽量包含这几个区域：

1. 教学区
2. 组合区
3. 危险区
4. 隐藏宝箱或支线区
5. 关底事件区

## 项目协作建议

如果多人开发，推荐按下面方式拆工：

- 玩法开发：角色控制、碰撞、道具、敌人
- 内容设计：地图、剧情、关卡事件
- UI 开发：HUD、菜单、章节界面
- 美术资源：角色、敌人、场景元素、图标
- 系统开发：存档、章节状态、配置数据
