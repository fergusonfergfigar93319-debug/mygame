# 背景剧情说明

## 世界背景

在遥远的冰海尽头，有一片被极光照亮的雪原。这里曾经不是赛道，而是小企鹅们的家园：雪松村负责点灯，冰湖回音谷负责传信，北境雾堤守着通往外海的古老冰桥。

咕咕嘎嘎和伙伴们每天在雪原上滑行、捕鱼、修灯塔。她们相信极夜并不可怕，只要伙伴还在，雪地里总会有回家的路。

直到高松鹅带着“静默风暴”袭来。

## 反派：高松鹅

高松鹅是一只冷酷、偏执、擅长布置冰原机关的坏蛋企鹅。他不喜欢雪松村的热闹，也不相信伙伴之间的自由协作。他认为极地世界应该只剩下命令、秩序和服从。

为了建立自己的冰原统治，他摧毁雪松村，抢走灯塔核心，把原本连接各地的冰道改造成危险赛道。裂谷、低矮冰拱、薄冰履带、风雪低能见区，都是他用来阻断救援的陷阱。

## 主角：咕咕嘎嘎

咕咕嘎嘎是一只勇敢、活泼、很重感情的小企鹅。家园被毁后，她没有选择躲起来，而是把碎掉的围巾重新系好，踏上寻找伙伴和追击高松鹅的旅途。

新的玩法方向里，咕咕嘎嘎会不断向前快跑。她不是为了比赛而跑，而是在和风暴抢时间：跑得越远，就越接近失散伙伴，也越接近高松鹅留下的真相。

## 伙伴与能力

- 团团：第一位救回的伙伴，擅长短时间掩护，可以在危险时帮咕咕嘎嘎撑过一段压力。
- Dror：同行伙伴，负责在旅途中辨认旧路标和隐藏线索，后续可扩展为发现彩蛋与剧情残页。
- 后续伙伴：建议分别绑定一种跑酷能力，例如破冰、照明、二段补救、短距离牵引。

## 跑酷道具的剧情解释

- 小鱼干：雪松村留下的能量补给，吃下后短时间冲刺。
- 泡泡围巾：旧灯塔的防护纤维，能让下落变得更柔和。
- 雪壳护盾：冰湖里的结晶外壳，可抵挡一次撞击。
- 风种：回音谷的风铃种子，能让跳跃更轻、更远。
- 极光磁针：Dror 找到的旧航具，会吸引鱼干、信标和残页。
- 微光残页：高松鹅行动计划的碎片，也是图鉴和章节记录的重要来源。

## 章节结构

### 第一章：雪松村废墟

咕咕嘎嘎从被毁的村庄出发，学习基础移动、跳跃和补给规则，救回团团。主题是“从失去中重新站起来”。

### 第二章：冰湖回音谷

冰湖会把声音传得很远，也会把错误放大。这里适合引入薄冰、风雪、低矮冰拱和更密集的道具选择。主题是“听见伙伴留下的线索”。

### 第三章：北境雾堤

雾堤是高松鹅最接近的前线，赛道开始出现 Boss 压迫感。高松鹅不一定总是正面出现，可以先用背景剪影、冰裂冲击波和封路机关制造存在感。主题是“穿过雾，看见真正的敌人”。

## 长期主线

1. 咕咕嘎嘎救回伙伴，修复雪原通讯。
2. 每个伙伴解锁一种新能力或新的赛道解法。
3. 玩家通过每日挑战和无尽快跑收集残页，逐步拼出高松鹅计划。
4. 最终所有伙伴汇合，在极光灯塔前阻止高松鹅启动静默风暴。

## 情绪基调

剧情应该保持温暖、轻快、有冒险感。危险存在，但不要沉重压抑；失败可以重来，伙伴会继续鼓励玩家向前跑。核心不是“打败一切”，而是“把回家的路重新跑出来”。
